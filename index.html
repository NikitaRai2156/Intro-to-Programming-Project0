Lesson 1: THE BASICS OF THE WEB AND HTML

How the Web Works

The web is fundamentally a group of computers communicating with each other. Whenever a person opens a web page, the computer sends an HTTP request to a server, which then finds the required HTML document and sends it back to the user's computer. Upon the completion of this action, the web browser that the user is using, interprets the web page and displays it on the screen of the output device that the user is using, whether it be a phone or a computer.

HTML stands for HyperText Markup Language. The majority content of the web is formed of HTML documents, which contain text elements that form what you see, and markup elements, that shape up how the page looks.

Why Computers are Stupid

Computers are simply machines, and thus they interpret every instruction literally. This means that a small mistake made by a programmer, when implemented literally by the computer, can cause a great deal of trouble.

Inline Elements Vs Block Elements

HTML elements can be either of the two types: inline or block. The block elements form an invisible 'block' around the content that they hold. <span>,<b>,<img> etc. are examples of inline element and <div>,<p> etc. are examples of block elements.
